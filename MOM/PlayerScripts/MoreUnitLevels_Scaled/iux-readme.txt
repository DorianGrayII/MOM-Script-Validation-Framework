This Mod increases the maximum number of levels a unit can achieve to 20.

After installation, the following was previously documented to be required.

1. Delete or Rename the following files: 
- steamapps\common\Master-Of-Magic\ExternalAssets\powerTrainingData.bin
- steamapps\common\Master-Of-Magic\ExternalAssets\powerTrainingData.txt
- steamapps\common\Master-Of-Magic\ExternalAssets\powerTrainingData_Short.txt

2. Start the game. Game will generate the powerTrainingData.bin file which may take a few minutes on the first startup.

3. Be patient while MoM is regenterated the powerTrainingData files, as it could take upto 10 minutes.
